# Tutorial

1. Clone repository
```bash
git clone git@github.com:fcocaceresc/todo-client.git
```

2. Fill .env_template
3. Change .env_template name to .env
```bash
mv .env_template .env
```
4. Create virtual environment
```bash
python3 -m venv venv
```
5. Install libraries
```bash
pip install -r requirements
```
6. Run the program
```bash
python3 main.py
```
